Transition Effect

Dynamic, Screenshot, Seamless

NFS Heat insipred transition effect. This script renders the screen on a texture, splits it into pieces relative to the screen size and prints them on the screen. Than removes them one-by-one (user dependent) without an additional animator etc. to present a seamless transition between scenes or takes. Can be used on start/finish loading scenes or on start/finish cutscenes.

Destructible Props

Physics, Progress,
Material Propery Block,
Save/Load

This system keeps the progress of broken props in order to present achievements or in-game discounts. It doesn't trigger if prop is already destructed/collected.
